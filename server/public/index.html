Documentación de API para Recuperación de Contraseña

Esta documentación cubre los endpoints utilizados para la recuperación de contraseña en nuestra aplicación.

Endpoint: Olvidé mi Contraseña (forgotPassword)

Método HTTP: POST

URL: /api/forgot-password

Cuerpo de la solicitud: { "email": "usuario@example.com" }

Descripción del flujo:

  • Validación de la solicitud y búsqueda del usuario.
  • Generación y envío del token de restablecimiento por correo electrónico.
  • Respuesta indicando el éxito del envío del correo.

Ejemplo de uso:

curl -X POST -H "Content-Type: application/json" -d '{"email": "usuario@example.com"}' http://localhost:5000/api/forgot-password

Endpoint: Restablecer Contraseña (resetPassword)

Método HTTP: POST

URL: /api/reset-password/:token

Cuerpo de la solicitud: { "newPassword": "nuevaContraseña123" }

Descripción del flujo:

  • Validación del token y actualización de la contraseña.
  • Limpieza de datos de restablecimiento de la contraseña.
  • Respuesta indicando el éxito del restablecimiento de la contraseña.

Ejemplo de uso:

curl -X POST -H "Content-Type: application/json" -d '{"newPassword": "nuevaContraseña123"}' http://localhost:5000/api/reset-password/abc123token
